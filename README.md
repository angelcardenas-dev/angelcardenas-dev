### 🛠️ Proyectos Destacados en Desarrollo

#### 💇‍♂️ Gavidia Salon Spa - Sistema Web Full-Stack Premium
Plataforma web comercial e integral optimizada para un prestigioso centro de estética en Miraflores, Lima.
* **Arquitectura de Vanguardia:** Desarrollado como una aplicación full-stack moderna y fuertemente tipada para garantizar un rendimiento óptimo y mantenibilidad.
* **UI/UX de Alto Nivel:** Interfaz responsiva premium con arquitectura de diseño híbrida (High-Contrast Dark Mode para la sección principal y Light Mode para interacción y geolocalización), construida con estilos altamente eficientes.
* **Funcionalidades Clave:** Catálogo dinámico de servicios, base de datos en la nube para gestión de información, pasarela de contacto directa y flujos optimizados de agendamiento de citas mediante integraciones modernas.
* **Stack Tecnológico:**
  * **Frontend:** Next.js (App Router), Tailwind CSS, TypeScript
  * **Backend & Base de Datos:** Node.js, Supabase (PostgreSQL / Auth)
  * **Calidad & Despliegue:** ESLint, Git, Vercel

#### 🎫 Startup: E-commerce & FinTech para Eventos Masivos (LatAm)
Cofundador y Líder Técnico de una plataforma bilateral diseñada para digitalizar la economía informal en conciertos, festivales y ferias en Perú y la región.
* **Lado Vendedor:** Gestión simplificada de inventario, mapas inteligentes de zonas de alta demanda y procesamiento de cobros digitales con lógica offline-first.
* **Lado Asistente:** Descubrimiento interactivo de feriantes, mapas internos de venues y pasarela ágil de entradas.
